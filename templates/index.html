Secret Garden Text Generator

How It Works

This application demonstrates text generation using machine learning techniques, specifically Recurrent Neural Networks (RNNs) with Long Short-Term Memory (LSTM) cells. Here's what happens:

  1. Training: The model learns from "The Secret Garden," identifying patterns, word sequences, and contextual relationships.
  2. Seed Text: A starting phrase is provided to guide generation.
  3. Text Generation: The model predicts the next word based on the seed text and its learned patterns.
  4. Temperature: Adjusts creativity:
    • Low (0.3): Predictable and coherent.
    • High (1.5): More random and creative.
